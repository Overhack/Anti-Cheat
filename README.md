# OFFICIAL OVERHACK ANTI-CHEAT API
Welcome to the official GitHub repository that will allow you to create your own **Anti-Cheat** System, thanks to the **Overhack Anti-Cheat API**. **Overhack Project** is a free competitive platform, with a dedicated Anti-Cheat, for the first time as a server-side application.
If you want to know more about **Overhack Project** please [click here](https://www.overhack.cloud).
